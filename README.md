# Kinetika

Фитнес-платформа на чистом HTML/CSS/JS. Без фреймворков, без сборки, без зависимостей — открываешь `index.html` и всё работает.

---

## Быстрый старт

```
form-studio/
├── index.html
├── style.css
├── app.js
└── assets/
    ├── banners/
    ├── programs/
    ├── products/
    └── ui/
```

Скопируй папку на любой веб-сервер или просто открой `index.html` в браузере. Для полноценной работы картинок нужен HTTP-сервер (из-за ограничений CORS на `file://`). Простейший вариант:

```bash
# Python
python -m http.server 8000

# Node
npx serve .
```

## Что умеет

### Авторизация
Экран входа/регистрации при старте. Поддерживает режим гостя. Имя пользователя подставляется в сайдбар, профиль и приветствие. Аккаунт появляется только после входа — до этого интерфейс заблокирован.

### Программы тренировок
8 авторских программ: Юкио Мисима, Майк Ментцер, Tren Twins, Ронни Колман, набор массы, похудение, Дэвид Гоггинс, Ислам Махачев. Каждая программа содержит полное расписание на 7 дней с упражнениями, фазовую разбивку по неделям, рекомендации по питанию и добавкам. Фильтры по категориям: Сила / Гипертрофия / Кардио / MMA / Выносливость.

### Онбординг — программы
3 шага: цель → уровень → количество дней в неделю. После завершения программы сортируются по релевантности, подходящие помечаются «Рекомендуем».

### Моя программа
Трекер тренировок с переключением между неделями. Прогресс-бар считает реальный процент выполненных тренировок за всю программу. Когда неделя завершена — предлагает перейти к следующей. Вкладка «История» показывает все прошлые недели с детализацией по дням.

### Питание
**Сегодня** — журнал приёмов пищи с группировкой по времени (завтрак, обед, перекус и т.д.). Блок КБЖУ всегда в одну строку — 4 карточки.

**Рацион на неделю** — генератор плана питания на 7 дней. Учитывает тип питания и бюджет из онбординга. Три уровня бюджета дают разные планы: эконом (яйца, гречка, скумбрия), средний (сбалансированный), премиум (стейки, лосось, морепродукты). Кнопка «Добавить весь день» переносит все приёмы пищи дня в журнал одним нажатием.

**Калькулятор** — формула Mifflin-St Jeor. Считает базовый обмен, TDEE, дефицит и профицит. Результат применяется как цель — автоматически пересчитывает нормы БЖУ.

**База продуктов** — 116 позиций по категориям: белки, углеводы, молочное, овощи, фрукты, жиры, спортивное питание, готовые блюда. Поиск, фильтр по категории, выбор граммовки.

### Онбординг — питание
3 шага: тип питания (стандарт / высокобелковое / веганское / кето) → калорийность (дефицит / поддержание / профицит / авто) → бюджет (до 400 ₽ / 400–800 ₽ / без ограничений). После завершения автоматически генерируется рацион на неделю и обновляются макросы.

### Магазин добавок
6 продуктов: протеин, предтрен, магний, омега-3, креатин, экстракт вишни. При активной программе показывает персональные рекомендации из неё. Корзина с управлением количеством, итоговой суммой и оформлением заказа.

### Рейтинг
Общий лидерборд с позицией, очками и динамикой. Баннер-герой для вставки своих данных — имя, место, статистика.

### Профиль
Аватар (загрузка с устройства), личные данные, статистика. Данные из профиля подтягиваются в калькулятор калорий.

## Кастомизация

### Поменять своё имя в рейтинге
В `index.html` найди блок `rating-hero-content` и замени имя, место и статистику:

```html
<div class="rating-hero-name">Твоё <em>Имя</em></div>
<div class="rating-hero-stat-val">#3</div>   <!-- место -->
<div class="rating-hero-stat-val">4 210</div> <!-- очки -->
```

### Добавить программу
В `app.js` в массиве `PROGRAMS` добавь объект по образцу существующих. Обязательные поля: `id`, `cat`, `title`, `desc`, `long`, `weeks`, `freq`, `schedule`, `blocks`, `stats`, `tags`, `obGoal`, `obLevel`, `obDays`. Добавь фото в `assets/programs/` и путь в `PROG_PHOTOS`.

### Добавить продукт в магазин
В `app.js` в массиве `PRODUCTS` добавь объект:

```js
{id:'myproduct', name:'Название', brand:'Бренд', price:1500, priceStr:'₽1 500', unit:'/уп.', desc:'Описание.'}
```

Добавь фото в `assets/products/` и путь в `PROD_PHOTOS`.

### Добавить продукт в базу питания
В `app.js` в массиве `FOOD_DB`:

```js
{id:'x1', name:'Название', cat:'protein', ico:'🍗', kcal:165, p:31, c:0, f:3.6, budget:'low'}
```

Категории: `protein`, `dairy`, `carbs`, `veg`, `fruit`, `fat`, `sport`, `meal`. Бюджет: `low`, `medium`, `high`.

### Цветовая схема
Все цвета — в CSS-переменных в начале `style.css`. Основные:

```css
--bg: #e8e4de;       /* фон приложения */
--card: #faf8f5;     /* карточки */
--ink: #18170f;      /* акцентный тёмный */
--amber: #b86c1a;    /* янтарный */
--sage: #4a7256;     /* зелёный */
```
